# Ordinal

Converts a number (e.g. 1) to an ordinal number (e.g. 1st)

## Installation

Add this line to your application's Gemfile:

    gem 'ordinal'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install ordinal

## Usage

	0.to_ordinal	# 0
	1.to_ordinal	# 1st
	5.to_ordinal	# 5th
	23.to_ordinal	# 23rd
